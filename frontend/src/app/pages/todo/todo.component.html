Mithilfe von Components können Bestandteile des Frontends ausgekapselt und in anderen Components wiederverwendet werden. Die Components können dann mithilfe dem (in der jeweiligen .component.ts Datei) definierten selector ganz einfach als HTML Komponente eingebunden werden. Bitte verwendet entweder den ng generate Befehl oder das Kontextmenü von Visual Studio Code zur Erstellung neuer Komponenten, da die Komponente auch in der app.module.ts Datei eingetragen werden muss!
